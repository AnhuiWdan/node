# node

设计一个简单的购物网站

clone下载本页代码到本地
cd node
npm install
node app #启动

选择的工具：
    "body-parser": "^1.18.2", //HTTP请求体解析中间件
    "express": "^4.16.2",     // web开发框架，基于node.js
    "express-session": "^1.15.6", //session中间件
    "mongoose": "^4.13.7",        // 操作mongodb
    "ejs": "^2.5.7"               // 模板
